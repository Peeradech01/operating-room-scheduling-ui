# 🏥 การจัดการกระบวนการในห้องผ่าตัดด้วยการจัดตารางเวลาและประสิทธิภาพที่เหมาะสม

## **(Optimized Scheduling and Efficiency in Operating Room Management)**

## 📖 บทนำ

โปรเจกต์นี้เป็นส่วนหนึ่งของรายวิชา โครงงาน  หลักสูตรวิทยาศาสตรบัณฑิต สาขาเทคโนโลยีสารสนเทศ คณะเทคโนโลยีสารสนเทศ สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง ปีการศึกษา 2568

ห้องผ่าตัดถือเป็นทรัพยากรที่มีค่าและมีจำกัดในโรงพยาบาล โดยมีต้นทุนการดำเนินงานสูง ทั้งค่าอุปกรณ์ทางการแพทย์ โครงงานนี้จึงมีวัตถุประสงค์เพื่อพัฒนาระบบในการจำลองการจัดตารางเวลาห้องผ่าตัดและเปรียบเทียบประสิทธิภาพอัลกอริทึม 3 รูปแบบ ได้แก่

| อัลกอริทึม                            | คำอธิบาย                                          |
| ------------------------------------- | ------------------------------------------------- |
| **ST(w, DEC, FF)**                    | วิธีการจัดตารางแบบฮิวริสติกที่ใช้กฎลำดับความสำคัญ |
| **Genetic Algorithm**                 | เจเนติกอัลกอริทึม                                 |
| **Genetic Algorithm with Q-learning** | เจเนติกอัลกอริทึมร่วมกับคิวเลิร์นนิง              |

## 📊 ตัวชี้วัดที่ใช้ประเมินผล

- **Penalty Score** — คะแนนบทลงโทษรวม (ยิ่งต่ำยิ่งดี)
- **จำนวนวันที่ใช้** — จำนวนวันที่ใช้จัดตารางทั้งหมด
- **Optimality Gap (%)** — ช่องว่างจากขีดจำกัดล่างทางทฤษฎี
- **Runtime (วินาที)** — ระยะเวลาที่ใช้ในการประมวลผล

👉 **[คลิกเพื่อเข้าใช้งานระบบจำลองการจัดตารางเวลาห้องผ่าตัด](https://operating-room-scheduling.streamlit.app/)**

### 💻 2. รันบนเครื่องส่วนตัว

**ขั้นตอนที่ 1 — โคลนโปรเจกต์**

```bash
git clone https://github.com/Best0129/operating-room-scheduling-ui.git
cd operating-room-scheduling-ui
```

**ขั้นตอนที่ 2 — สร้าง Virtual Environment**

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

**ขั้นตอนที่ 3 — ติดตั้งไลบรารีที่จำเป็น**

```bash
pip install -r requirements.txt
```

**ขั้นตอนที่ 4 — รัน Streamlit**

```bash
streamlit run app.py
```
